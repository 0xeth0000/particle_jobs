## 任务执行方式

- [ ] 1.在data目录下找到data.xlsl，填写你自己的twitter_token,dc_token,proxy,key

- [ ] 2.在config配置你的打码平台信息和一些配置项

    安装python,建议python3.11,打开cmd或shell

  1. `cd /particle_jobs`
  2. `python -m venv venv`
  3. `venv\Scripts\activate`
  4. `pip install -r requirements.txt`
  5. `python main.py`
